# Superstore Power BI & SQL Project

An end-to-end data analytics project using the Sample Superstore dataset. This project includes data cleaning using SQL, analysis, and an interactive Power BI dashboard.

## Project Structure

- `dataset/` – Superstore CSV file  
- `sql/` – SQL scripts for cleaning and analysis  
- `dashboard.pbix` – Power BI dashboard file  
- `screenshots/` – Dashboard images  
- `documentation/` – Project notes  

## Tools Used

- SQL (MySQL / SQL Server)
- Power BI
- Excel / CSV

## Project Workflow

1. Load dataset (Superstore CSV)
2. Clean data using SQL
3. Perform analysis using SQL queries
4. Build dashboard in Power BI
5. Extract insights

##  Dashboard Includes

- Sales & Profit KPIs
- Regional performance
- Category & sub-category analysis
- Trend analysis over time

##  Key Insights

- Identified top-performing regions and products  
- Analyzed profit vs discount relationship  
- Found best and worst performing segments  

##  How to Use

1. Clone the repo  
2. Open SQL scripts and run queries  
3. Open `dashboard.pbix` in Power BI Desktop
